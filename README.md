# Python Standard Libraries Notebooks

This repository contains a collection of Jupyter Notebooks serving as a comprehensive reference guide to Python's standard libraries. These resources are designed to provide practical examples and deep dives into essential Python modules.

## Contents

The notebooks are organized as follows:

1.  **[Notebook 01 - Strings and Encodings](Notebook%2001%20-%20Strings%20and%20Encodings.ipynb)**
    *   Deep dive into string manipulation, formatting, and encoding standards.

2.  **[Notebook 02 - Math Library Reference](Notebook%2002%20-%20Math%20Library%20Reference.ipynb)**
    *   Overview of the `math` module capabilities.

3.  **[Notebook 03 - OS,Sys,Time Reference](Notebook%2003%20-%20OS,Sys,Time%20Reference.ipynb)**
    *   Working with operating system interfaces (`os`), system-specific parameters (`sys`), and time handling (`time`).

4.  **[Notebook 04 - Regex Library Reference](Notebook%2004%20-%20Regex%20Library%20Reference.ipynb)**
    *   Comprehensive guide to regular expressions using the `re` module.

5.  **[Notebook 05 - Socket Library Reference](Notebook%2005%20-%20Socket%20Library%20Reference.ipynb)**
    *   Networking basics and socket programming in Python.

6.  **[Notebook 06 - Urllib Library Reference](Notebook%2006%20-%20Urllib%20Library%20Reference.ipynb)**
    *   Handling URLs and making HTTP requests using `urllib`.
